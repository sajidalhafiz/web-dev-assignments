What is the differences between map(), forEach(), filter() and find()?

  • map(): It is an array method used to return a new array after performing some operations on the given array.
  • forEach(): It works same as map() method but the only difference is forEach() method does not return after performing operations.
  • filter(): It gives a new array from the given array containing all the values that passes the condition.
  • find(): It only gives the first item that matches the condition from the given array.

What is the differences between localStorage, sessionStorage and cookies?

  • localStorage: It is used to store large amount of user's data in the browser where data is saved after closing the browser.
  • sessionStorage: It is used to store data for user's preference where data is not saved after closing the browser.
  • cookies: It is same as sessionStorage but has less capacity of data storage.

What is Event-loop? How it works?

The Event-loop is a mechanism that works on browser API when asynchronous code is need to be performed.

For example: Consider, there are 3 console.log method called for showing 3 numbers from 1 to 3 and in between the console.log of 2 and 3 there is an asynchronous function fetch is called. The program will begin to execute by calling each line of code in the call-stack. Synchronous code will execute from the call-stack but asynchronous waits for Promise and stored sequentially in the queue. Then again it goes to the call-stack and runs the code.
