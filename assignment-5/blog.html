Questions & Answers

Q-1: What is DOM? What is the purpose of DOM?

DOM - Document Object Model is actually a programming interface where we can manipulate the webpage structure, style, and content. The purpose of DOM is to access and manipulate HTML documents in a browser using a scripting language such as JavaScript.

Q-2: How to select DOM elements?

There are a few ways to select DOM elements:

  1. getElementById(): The method access the element by using the element id.
  2. getElementByClassName(): The method access the collection of elements by using the class name of the elements.
  3. getElementByTagName(): The method access the element using the element tag name.
  4. querySelector(): The method selects the first element from the document when it is called using the class name.
  5. querySelectorAll(): The method returns a collection of selected elements.

Q-3: What is event bubble?

Event bubble is a concept used in DOM. When an event handler is set to an element, the event triggers that element and bubbles the event to its parent element.

It has three phases:

  1. Capture phase: In this phase, event is captured in the target element.
  2. Target phase: Here, performs the event of the target element and starts bubbling.
  3. Bubble phase: If there are events in the parent element, the bubbling phase performs the events.

Q-4: What is the callback function and its uses?

When a function is passed as an argument of another function is the callback function. It is used where one function should wait for the other function.
